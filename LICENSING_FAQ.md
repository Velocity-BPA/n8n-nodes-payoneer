# Licensing FAQ

## General Questions

### What license does this n8n node use?
This node is licensed under the Business Source License 1.1 (BSL 1.1).

### Can I use this for free?
Yes, you can use this node free of charge for:
- Personal projects
- Educational purposes
- Research
- Internal business use (non-commercial)

## Commercial Use

### Is internal SaaS infrastructure considered internal use?
No. If the infrastructure supports a revenue-generating service, it is considered commercial use.

### What if customers never see the node?
Visibility is irrelevant. Revenue enablement constitutes commercial use.

### What if we are a startup?
Startups are commercial entities and require a commercial license if using this node in production for commercial purposes.

### What if we only use one workflow?
Scope does not change license requirements. Even a single workflow used commercially requires a license.

### Do proof-of-concepts require a license?
Internal, non-customer-facing POCs are permitted. Production SaaS deployments require a license.

### What about consulting or managed services?
If you're using this node as part of a paid engagement for clients, you need a commercial license.

## Technical Questions

### Does the license affect how the node works?
No. The node functions identically regardless of license type. Licensing is purely a legal matter.

### Is there any telemetry or license enforcement code?
No. We use an honor-based system. The license notice is informational only.

### Can I modify the code?
Yes, you can modify the code for your own use. However, redistribution of modified versions must comply with BSL 1.1 terms.
